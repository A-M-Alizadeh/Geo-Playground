Understanding Ranging Codes

Purpose: Ranging codes are pseudorandom sequences that allow GPS receivers to measure the time it takes for signals to travel from satellites, enabling distance calculation.

Properties: Good autocorrelation (sharp peak at zero lag), low cross-correlation with other codes, and uniform spectral distribution.

C/A Code: 1023-chip Gold code sequence transmitted at 1.023 Mcps, repeating every 1 millisecond.

Understanding BOC Modulation

Purpose: BOC modulation splits the signal spectrum, reducing interference with existing GPS signals while improving ranging accuracy.

BOC(m,n): m = subcarrier frequency / 1.023 MHz, n = code rate / 1.023 Mcps

Select a BOC type to see specific details...

Advantages: Better multipath resistance, improved cross-correlation properties, and compatibility with legacy systems.

Split Spectrum: Creates two main spectral lobes separated by the subcarrier frequency, reducing interference.

Signal Types:

  • Galileo E1: BOC(1,1) for Open Service, BOC(6,1) for PRS
  • GPS L1C: BOC(10,5) for improved performance
  • Galileo E5: AltBOC for wideband signals
  • MBOC: Multiplexed BOC combining different subcarriers
